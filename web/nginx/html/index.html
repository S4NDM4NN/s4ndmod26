Quick Install

Downloads iortcw client, base game paks, and the mod. Run one command for your OS.

🐧 Linux
Open a terminal and run:
click to copy
Then launch:
~/rtcw/iowolfmp.x86_64 +set fs_basepath ~/rtcw +set fs_game s4ndmod26 +connect s4ndmod.com click to copy

Installs to ~/rtcw/. Override with RTCW_DIR=~/games/wolf curl …

🪟 Windows
Open PowerShell and run:
click to copy
Then launch:
~\wolf\ioWolfMP.x64.exe +set fs_basepath ~\wolf +set fs_game s4ndmod26 +connect s4ndmod.com click to copy

Installs to %USERPROFILE%\wolf\. Override with $env:RTCW_DIR="C:\Games\wolf" before running.
